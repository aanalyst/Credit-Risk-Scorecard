# Credit-Risk-Scorecard

| |
|---|
| **Background** |
| Australian banks and lenders are required to assess borrower creditworthiness before approving loans. Without a structured, data-driven approach, lenders expose themselves to significant credit losses — particularly from borrowers who appear low-risk but default unexpectedly. This project builds an end-to-end credit risk scorecard using **1,345,310 real loan records** from Lending Club (2007–2018), one of the largest peer-to-peer lending platforms. The dataset mirrors real-world lending patterns including borrower grades, income levels, debt obligations, and repayment outcomes. Reporting as a data analyst embedded in a credit risk function, an end-to-end analysis was conducted to identify default drivers, score borrower risk using logistic regression, and produce a points-based credit scorecard. The key insights and recommendations focus on the following areas: **North Star Metrics** — Default rate by grade, DTI, FICO score, and loan purpose; **Model Performance** — Logistic regression with class balancing and feature scaling; **Credit Scorecard** — Probability-to-points conversion using industry-standard PDO methodology; **SQL Risk Analysis** — SQLite-based risk band queries for decision support |

Dataset Source: Lending Club Loan Data (2007–2018Q4). Available on Kaggle: https://www.kaggle.com/datasets/wordsforthewise/lending-club
